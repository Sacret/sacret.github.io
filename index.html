Githubify

The beloved project of mine that started from the problem of finding some GitHub repo (your own or starred) when you forgot its title. It is kinda tricky, yay! But what about adding tags to repos for easy and fast search? Using Githubify.me you will have ability to set tags for your own repos, forks, or repositories where you participate as a collaborator and even starred repos! It was built using:

  • React
  • Reflux
  • Firebase
  • Superagent
  • Bootstrap and so on
